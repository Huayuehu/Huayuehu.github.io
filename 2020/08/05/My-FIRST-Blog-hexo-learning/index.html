My FIRST Blog -- hexo learning

  1. Required environment: Node.js npm

  2. Install hexo
    npm install -g hexo-cli

  3. Make a new directory for hexo blog
    mkdir blog
    cd blog

  4. Initialize hexo
    sudo hexo init

  5. Start hexo: hexo s
    Hexo will be running at http://localhost:4000 for default

  6. Make a new blog: hexo n "My FIRST Blog -- hexo learning"

  7. Generate that first blog and restart the server
    hexo clean // clean the database
    hexo g
    hexo s
